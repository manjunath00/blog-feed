# Buzz-feed

A web application where you can create articles & view articles created by others

## User Story
 
* A user can Create, Update his articles
* A user can subscribe to certain categories

## Installation

``` 
git clone <repo-url>
cd <repo-dir>
# install and run via npm or yarn
npm i && npm start
```
